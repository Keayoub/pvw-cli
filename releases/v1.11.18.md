# Release Notes - v1.11.18

**Release Date**: May 15, 2026
**Previous Version**: 1.11.17

## Overview

Unified Catalog term updates now preserve the existing server-returned payload shape, and `update-json` now supports `--debug` like the CSV path.

## Fixed

- Preserve existing term fields when updating glossary terms through Unified Catalog.
- Add `--debug` support to `uc term update-json` for parity with `update-csv`.
