# Release v1.3.8

## Nouveautés
- Option `--json` : sortie JSON brute, compacte, sans numérotation ni coloration (idéal pour scripts et PowerShell)
- Option `--json-detail` : sortie JSON pretty-print colorée (Rich), pour lecture humaine
- Extraction des colonnes de tables SQL simplifiée côté PowerShell (plus besoin de nettoyage)
- Correctifs robustesse parsing JSON dans les scripts d'extraction

## Améliorations
- CLI : meilleure distinction entre sortie machine et sortie interactive
- Documentation et exemples PowerShell mis à jour

## Migration
- Pour automatisation, utilisez désormais `--json` dans vos scripts
- Pour inspection manuelle, utilisez `--json-detail`
