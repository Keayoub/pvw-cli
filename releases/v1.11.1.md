# Release Notes - v1.11.1

**Release Date**: May 1, 2026  
**Previous Version**: 1.11.0

---

## Overview

v1.11.1 focuses on bulk entity operations hardening and documentation usability.

This release includes bulk CSV retry/throttle controls, practical command presets,
and a major docs navigation cleanup to make high-frequency workflows easier to find.

---

## Highlights

### 1. Bulk Entity CSV reliability and throughput controls

The following options were added to bulk CSV create and update workflows:

- `--throttle-ms`
- `--max-retries`
- `--retry-backoff-ms`
- `--retry-mode` (`fixed` or `exponential`)

This improves stability under API throttling while keeping high-throughput tuning possible.

### 2. Improved bulk update behavior

`bulk-update-csv` now uses batch-style bulk payload processing for GUID-driven updates,
reducing slow per-row behavior and making runtime much more predictable on large files.

### 3. Better built-in command guidance

`bulk-create-csv --help` and `bulk-update-csv --help` now include:

- Fast profile
- Balanced profile
- Safe profile
- Quick tuning recommendations

### 4. Documentation navigation refresh

New practical pages were added and linked into top-level docs navigation:

- Common workflows entry point
- Task pages by operation (create, update, delete, import)
- Bulk CSV troubleshooting page
- Command decision tree page
- Start-here sections at the top of large catalog pages
