## ✨ New Features

### Phase 3: Advanced Entity & Lineage Operations

**Entity Commands** (4 new):
- `pvw entity history` - View entity change history and audit trail
- `pvw entity validate` - Validate entity structure and metadata
- `pvw entity dependencies` - Analyze entity dependencies and impact
- `pvw entity usage` - Get entity usage statistics and patterns

**Lineage Commands** (3 new):
- `pvw lineage upstream` - Trace upstream data sources
- `pvw lineage downstream` - Identify downstream data consumers
- `pvw lineage temporal` - Analyze lineage changes over time

### Phase 4: Analytics & Reporting

**Account Analytics** (3 new):
- `pvw account analytics` - Comprehensive account metrics (entity counts, collections, storage, activity)
- `pvw account usage` - Resource usage statistics (API calls, storage, scans, connections)
- `pvw account limits` - Resource limits and quotas

**Collection Analytics** (1 new):
- `pvw collections analytics` - Asset counts, hierarchy metrics, usage patterns
- `pvw collections analytics --collection-name <name>` - Collection-specific analytics

**Search Analytics** (1 new):
- `pvw search analytics` - Query patterns, popular assets, search trends, performance metrics

### 100% Unified Catalog API Coverage

**Final APIs Completed**:
- `pvw uc term delete-relationship` - Delete specific term relationships
- All Facets APIs (4): term, dataproduct, cde, objective facets
- Hierarchy API (1): term hierarchy visualization
- Relationships APIs (3): list, add, delete term relationships

## 🎯 Use Cases Enabled

### Advanced Data Lineage
```bash
# Trace data lineage upstream and downstream
pvw lineage upstream --guid <entity-guid> --depth 5
pvw lineage downstream --guid <entity-guid> --depth 5

# Analyze temporal lineage changes
pvw lineage temporal --guid <entity-guid> --start-time 2026-01-01T00:00:00Z
```

### Entity Management
```bash
# View entity change history
pvw entity history --guid <entity-guid>

# Validate entity structure
pvw entity validate --guid <entity-guid>

# Analyze dependencies
pvw entity dependencies --guid <entity-guid>
```

### Analytics & Insights
```bash
# Account-level analytics
pvw account analytics
pvw account usage
pvw account limits

# Collection analytics
pvw collections analytics
pvw collections analytics --collection-name my-collection

# Search patterns
pvw search analytics
```

### Unified Catalog Relationships
```bash
# List all term relationships
pvw uc term relationships --term-id <term-guid>

# Delete obsolete relationships
pvw uc term delete-relationship --term-id <term-guid> --entity-id <entity-guid>

# View glossary hierarchy
pvw uc term hierarchy --max-depth 5
```
