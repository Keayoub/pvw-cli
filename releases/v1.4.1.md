# Version 1.4.1 - Bulk Create + UC Term Custom Attributes

Release Date: 2026-01-08  
Type: Minor Feature + Fixes

## Highlights
- Entity bulk-create from CSV now supports `--debug` and nested custom attributes via dot notation (e.g., `customAttributes.sensitivity`, `businessMetadata.dept`).
- Unified Catalog term update from CSV supports `customAttributes.*` columns; values are merged with existing attributes.
- Dry-run previews improved to show parsed attributes and payloads for faster troubleshooting.

## Commands
- Entity bulk create (with debug/dry-run):
  ```bash
  pvw entity bulk-create-csv --csv-file path/to/entities.csv --dry-run --debug
  ```
- UC terms update with custom attributes:
  ```bash
  pvw uc update-terms-from-csv --csv-file path/to/terms.csv --dry-run --debug
  ```

## Notes
- Use global `--debug` before the main group if subcommand parsing conflicts occur.
- Backward compatible; no breaking changes.
