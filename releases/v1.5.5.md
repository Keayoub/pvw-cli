# v1.5.5 Release Notes

## Fixes

- Guarded CSV custom attribute parsing against empty/None headers to avoid `NoneType` errors.
- Improved handling of list values in `customAttributes.*` columns for term import CSV.
- Updated sample CSVs used for custom attributes with arrays.

## Testing

- CSV import verified on Windows PowerShell with list-valued custom attributes.
