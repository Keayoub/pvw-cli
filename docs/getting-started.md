# Getting Started

## Build Docs Locally

1. Generate catalogs:

```bash
python scripts/generate_mkdocs_catalog.py
```

2. Build the site:

```bash
mkdocs build --strict
```

3. Open the generated site:

- site/index.html

## Optional Live Preview

```bash
mkdocs serve
```

Then open http://127.0.0.1:8000
